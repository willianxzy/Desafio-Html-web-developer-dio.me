Falando sobre tags

Nessa aula foi ensinado sobre as tags do HTML.

  • Tag I

A primeira tag que foi ensinada foi sobre a tag i, na qual deixa o texto em itálico.

Exemplo: Itálico

  • Tag Strong

A tag Strong faz com que o texto fique em negrito, dando assim mais destaque para a palavra, frase ou citação que a pessoa quer que fique mais chamativa.

Exemplo: Negrito

  • Tag Input

Input é uma tag na qual ela tem atributos, e esses atributos conseguimos criar algumas coisas, dentre elas temos o input de texto, que cria uma caixa para o usuário conseguir digitar.

Exemplo: Qual o seu nome?

Ao abrir essa caixa de texto o usuário consegue digitar oque está pedindo.

Uma curiosidade sobre a tag input é que ela se auto fecha, não é necessário colocar o / na frente do nome da tag para ela se fechar.

  • Tag IMG

Outra tag que se auto fecha é a tag IMG, a tag img é nada mais nada menos que colocar uma imagem para ser vista pelo usuário.

Exemplo: Foto do Neymar

As tags input e img além delas, elas possuem atributos, esses atributos são diferentes funções que pode usar com elas para aprimorar ainda mais o seu código HTML.

Atributos das Tags

Nesta aula aprendemos sobre os atributos que as tags tem. Os atributo são propriedades que uma tag pode ter.

Existem atributos genéricos e atributos mais específicos para cada tag. Alguns atributos são: Id, style, class, src, width, dentre outros..

Textos

Textos é basicamente oque o usuário vai ver no website, essa parte de textos está associada a tipografia. Se for pesquisar sobre tipografia do HTML vai aparecer diversos projetos para se ter como exemplo.

  • Tag H

A primeira tag da tipografia é o h1, que significa header 1 ou cabeçalho 1. O Header tem níveis, ele é composto desde o h1 até o h6, que é como se fossem titulos, subtitulos e assim por diante.

  • Tag P

Outra tag que dá inicio ao corpo do texto é o P, o P é nada mais, nada menos que paragraf, ou Parágrafo. Toda vez que se finaliza o texto e coloca um ponto, pula uma linha e adiciona o P para fazer outro parágrafo.

  • Tag Blockquote

Temos também a tag blockquote, que ela é usada para colocar citações. Vamos colocar um exemplo de como funciona o blockquote.

Exemplo:

Está é a função do Blockquote!

Ou seja, ela é utilizada em citações e ainda abre uma margem do começo da página para ganhar mais destaque, mas lembrando que as letras em itálico não é função do blockquote, somente o modo como o texto fica no website

  • Tag Strong

A tag muito utilizada também é a tag strong, que faz com que o texto fique em negrito.

  • Tag U

A tag U é utilizada para sublinhar o texto, fazendo assim com que ele ganhe mais ênfase e destaque.

  • Tag Mark

Temos a tag Mark, a tag mark faz com que se crie uma marcação no texto, do mesmo modo como um marca texto convencional.

Temos uma tag que tem a mesma função de marcar como registrado, que é a tag sup, a tag sup faz com que as letras fique no topo.

  • Tag Sub

Temos também a sua irmã, que é a tag sub, que tem o mesmo principio, porém as letras ficam embaixo.

Vejamos: Limitededition

Listas Ordenadas e não ordenadas

As tags de listas ordenadas são descritas por ol e ul, OL= order list (listas ordenadas) e UL= Unordered List (lista não ordenada)

Para conseguirmos adicionar um texto dentros das listas precisamos acionar suas "filhas", que são as LI = List item (item da lista). Portante dessa maneira você poderá adicionar conteúdo a sua lista.

A diferenciação da lista ordenada para a lista não ordenada são que, ao criar uma lista ordenada ela adiciona números em frente aos itens da sua lista

Vamos para o Exemplo:

  1. Item X
  2. Item Y
  3. Item Z

Já a lista não ordenada ela coloca marcadores em frente ao item da lista

Vamos para o Exemplo:

  • Item X
  • Item Y
  • Item Z
